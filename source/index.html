Get Started

It's easy to get started. Just follow the steps below:

1. Download

Download espresso or clone from Github.


$ git clone git@github.com:kiriaze/espresso.git

2. Install Dependencies

Run Bower & Node Package Manager to install all project dependencies. 


$ bower install
$ npm install --global gulp && npm install

3. Configure

Enter the desired values in the _settings.scss file.
These are just a few of those settings.


$container-class: 'container';
$row-class: 'row';
$column-class: 'columns';
$number-of-columns: 12;
$gridDebug: true;
$gutter: 0;

4. Build

Run gulp


$ gulp

Basic usage

This is the default usage. The grid is calculated and classes are generated for you to get building quickly.

Advanced usage

By setting the $advanced-usage variable in _variables.scss to 'true' you can work with espresso more semantically.
